HTML Tags for Web Development

Basic Structure

  • <!DOCTYPE html> – Declares document type
  • <html> – Root element
  • <head> – Metadata
  • <body> – Page content

Text Formatting

  • <h1> to <h6> – Headings
  • <p> – Paragraph
  • <strong> – Bold text
  • <em> – Italic text
  • <code> – Inline code

Structure & Navigation

  • <a> – Anchor (hyperlink)
  • <img> – Image
  • <ul>, <ol>, <li> – Lists
  • <table>, <tr>, <td> – Tables

Forms

  • <form> – Form element
  • <input>, <label>, <textarea>
  • <button> – Button
