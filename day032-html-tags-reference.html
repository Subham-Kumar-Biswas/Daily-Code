Common HTML Tags

Text Tags

  • <h1> to <h6> - Headings
  • <p> - Paragraph
  • <strong> - Bold text
  • <em> - Italic text

Structure Tags

  • <div> - Division
  • <span> - Inline container
  • <hr> - Horizontal rule

List Tags

  • <ul> - Unordered list
  • <ol> - Ordered list
  • <li> - List item
