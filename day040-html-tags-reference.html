Common HTML Tags

Text Formatting

  • <p> – Paragraph
  • <h1> to <h6> – Headings
  • <strong> – Bold text
  • <em> – Italic text
  • <br> – Line Break

Lists

  • <ul> – Unordered List
  • <ol> – Ordered List
  • <li> – List Item

Links & Images

  • <a> – Anchor/Link
  • <img> – Image

Forms

  • <form> – Form container
  • <input> – Input field
  • <button> – Button
